필수 기능 )
 1. 식품명 검색
	-> 제품명, 제조사, 원재료 기준으로 검색 가능 
	-> 상세 검색시 영양소 함량을 그래표로 표시
 2. 식품영양정보관리
	-> 식품에 대한 영양정보 검색 (원재료 기준으로 검색 가능)
 3. 회원 관리
	-> 회원 기본 정보, 알레르기 정보 CRUD
 4. 로그인 관리
	-> 로그인 / 로그아웃
	-> 비밀번호 찾기

추가 기능 )
 1. 식품별 알레르기 표시
	-> 식품에 대한 알레르기 성분 표기 제공
	-> 사용자가 설정한 알레르기 성분이 존재하면 표시
 2. 섭취 식품 데이터 관리 (찜하기)
	-> CRUD
	-> 등록시 회원정보 알레르기 주의 경보 표시 -> 찜하기 불가
